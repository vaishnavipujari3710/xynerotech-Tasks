# AI World - Creative Landing Page

This is a modern, responsive landing page built with **Tailwind CSS**.  
It showcases AI World’s innovative solutions with smooth animations and mobile-friendly navigation.

## Features

- Responsive design using Tailwind CSS utility classes  
- Sticky header with desktop and mobile menu toggle  
- Fade-in scroll animations for a polished user experience  
- Clear sections: Home, About, Services, Contact  
- Interactive buttons and hover effects  
- Footer with social media links

## Demo

Open the `index.html` file in your browser to view the landing page.

## How to Use

1. Clone or download the repository.  
2. Open `index.html` in any modern web browser.  
3. Customize content and images as needed.

## Technologies Used

- HTML5  
- Tailwind CSS  
- Vanilla JavaScript for menu toggle and animations
